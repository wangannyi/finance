# 本地私有投资情报网站

## 隐私边界

本网站只运行在本机 `127.0.0.1:8765`，不发布公网，不接入妙搭或飞书平台。数据默认写入本仓库的 `data/reports.sqlite3`。

爬虫会访问公开财经网站和交易所/监管相关网站，因此你的电脑会产生正常的外网访问记录。不会主动上传你的本金、仓位或个人信息。

## 启动网站

```sh
scripts/run_dashboard.sh
```

浏览器打开：

```text
http://127.0.0.1:8765
```

## 手动更新数据

```sh
scripts/refresh_reports.sh
```

也可以在网页右上角点击“立即更新”。更新后页面会自动读取本地 SQLite 的最新结果。

## 每天自动更新

安装本地 cron 定时任务：

```sh
scripts/install_daily_cron.sh
```

默认每天早上 08:30 运行一次爬虫，日志写入：

```text
data/refresh.log
```

查看当前定时任务：

```sh
crontab -l
```

如果要取消，运行 `crontab -e` 删除包含 `scripts/refresh_reports.sh` 的那一行。

## 当前能力

- A 股、港股、美股三市场每日热点方向。
- 每个市场展示一天、一周、一个月三个时间维度。
- 每个方向包含热度评分、支撑证据、主要风险和代表龙头。
- 每个方向会显示命中的原始来源、标题和关键词，便于判断结论是否可靠。
- 美股科技方向拆分到 AI 存储/HBM、CPO/光通信等细分主题，避免只归入笼统 AI。
- 点击龙头公司可查看市值、PE、近 3 天、近 1 周、近 1 个月涨跌幅、52 周高低点等行情指标。
- 组合规划面板展示 50 万本金、A 股:港股:美股 = 3:1:1 的目标金额和新手风险护栏。
- 候选池会从三市场热点报告中抽取代表龙头，展示动作、风险标签、观察仓上限和买入前检查。
- AKShare 增强数据源安装后可显示 A 股/港股行情快照。
- 页面每 5 分钟自动从本地 API 重新读取数据。

## 注意事项

当前爬虫采用公开网页标题和关键词评分，适合做每日观察和线索聚合，不适合直接作为买卖依据。涉及具体买入前，仍需要人工核验公告、财报、估值和风险。

候选池仅用于观察和复盘，不代表买入建议。页面中的“观察仓上限”是风险护栏，不是建议买满的金额。

公司详情行情使用 Yahoo Finance 的公开 quote/chart 接口做跨市场查询。A 股 `.SH` 代码会自动转换为 Yahoo 使用的 `.SS` 后缀。若接口暂时不可用或字段缺失，页面会显示“暂无数据”或缓存数据。

如需启用 AKShare：

```sh
python -m pip install -r requirements.txt
```

AKShare 主要用于增强 A 股和港股数据。未安装时，网站不会崩溃，只会在“增强数据源”面板提示安装方式。
