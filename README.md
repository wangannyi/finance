# 私人投资情报看板

这是一个本地私有运行的投资研究辅助项目，用于每天跟踪 A 股、港股、美股的热门投资方向，并在网页上滚动展示更新结果。

项目面向投资新手，核心目标不是给出确定性买卖指令，而是帮助你每天整理市场线索、查看热点方向、比较代表龙头，并在做投资决策前明确风险和需要继续核验的数据。

## 功能概览

- 跟踪 A 股、港股、美股三个市场。
- 每个市场展示接下来一天、一周、一个月的 Top 5 热门方向。
- 每个方向包含热度评分、支撑证据、主要风险和代表龙头企业。
- 美股科技方向已细分为 AI 基础设施、AI 存储/HBM、CPO/光通信等更具体主题。
- 点击龙头公司可查看当前价格、PE、市值、近 3 天、近 1 周、近 1 个月涨跌幅等信息。
- 展示 50 万本金、A 股:港股:美股 = 3:1:1 的组合框架和新手风险护栏。
- 从热点报告自动生成候选池，并展示持仓前检查项、风险标签和观察仓上限。
- 可选接入 AKShare，增强 A 股和港股行情快照。
- 支持网页右上角手动刷新。
- 支持本机 cron 每天自动刷新。
- 数据保存在本地 SQLite，不发布公网，不接入妙搭或飞书平台。

## 快速开始

项目只依赖 Python 标准库即可运行基础看板。若要启用 AKShare 增强数据，先安装依赖：

```sh
python -m pip install -r requirements.txt
```

启动本地网站：

```sh
scripts/run_dashboard.sh
```

浏览器打开：

```text
http://127.0.0.1:8765
```

手动刷新三市场数据：

```sh
scripts/refresh_reports.sh
```

安装每天 08:30 自动刷新：

```sh
scripts/install_daily_cron.sh
```

查看定时任务：

```sh
crontab -l
```

## 测试

运行全部测试：

```sh
python -m unittest discover -s tests -v
```

测试覆盖报告生成、SQLite 存取、股票代码转换和区间涨跌幅计算。

## 隐私说明

网站只绑定本机地址 `127.0.0.1:8765`，默认不会被局域网或公网访问。项目不会主动上传你的本金、仓位或个人信息。

爬虫会访问公开财经网站和行情接口，因此你的电脑会产生正常外网访问记录。不要把本项目部署到公司平台或公开服务器，除非你确认访问权限和日志风险。

## 数据限制

当前热点方向主要基于公开网页标题、关键词和配置中的行业方向进行评分，适合做每日观察和研究线索聚合，不适合作为直接买卖依据。

美股热点源包含泛财经首页、科技新闻、半导体专题以及部分光通信/CPO 产业链专题源。若市场出现新主题，需要继续扩展 `finance_app/config.py` 中的主题词和来源。

公司详情优先使用 Yahoo Finance 的公开 chart 数据计算价格和区间涨跌幅。部分市值、PE 等 quote 字段可能因接口限制显示“暂无数据”。具体投资前仍需核验交易所公告、财报、估值和风险。

组合规划中的单标的上限、单主题上限和分批次数是新手默认护栏，不是最终投资建议。真正买入前仍需确认投资期限、最大可承受回撤、现金储备和账户条件。

候选池来自每日热点报告中的代表龙头，作用是帮助你组织观察对象和买入前检查，不是自动买入清单。

AKShare 是可选增强数据源，主要用于 A 股、港股和中国金融市场数据。若未安装，网站会继续运行并显示启用提示。

更多本地运行说明见 [docs/local-private-dashboard.md](docs/local-private-dashboard.md)。
